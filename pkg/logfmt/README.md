## ANSI escape code 语法颜色笔记

http://noyobo.com/2015/11/13/ANSI-escape-code.html

- 30-37 设置文本颜色
    * black: 30
    * red: 31
    * green: 32
    * yellow: 33
    * blue: 34
    * magenta: 35
    * cyan: 36
    * white: 37
- 40–47 设置文本背景颜色
- 39 重置文本颜色
- 49 重置背景颜色
- 1 加粗文本 / 高亮
- 22 重置加粗 / 高亮
- 0 重置所有文本属性（颜色，背景，亮度等）为默认值
